# Retail Sales Analytics Dashboard for Krishna Store

## Overview
Built an interactive Excel dashboard analyzing 31K+ retail transactions to identify customer purchasing behavior, sales trends, and channel performance.

## Features
- Data Cleaning & Processing
- Pivot Tables & Pivot Charts
- Interactive Slicers
- KPI-Based Dashboard
- Customer Segmentation Analysis
- Business Insights & Recommendations

## Key Insights
- Women customers aged 30–49 contributed the highest sales share
- Maharashtra, Karnataka, and Uttar Pradesh were top-performing states
- Amazon, Flipkart, and Myntra generated nearly 80% of total channel sales

## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization
- Dashboard Reporting
